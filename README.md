# Eve-Theme
Re-create EVE Online Launcher controls in .NET, a Total of 23 custom controls added.

# How to use:
 + Download latest build here: https://github.com/qiangqiang101/Eve-Theme/releases
 + Add EveTheme.dll to reference
 + Build your project
